TauViz-CT

TauViz-CT is a lightweight demo tool for visualizing 3D porous microstructures with speed-colored streamlines derived from v = -∇C (a pseudo-flow field from a 3D concentration map). The rendering overlays a semi-transparent solid skeleton and optional “comet-tail” glow points, producing an intuitive “flowing inside pores” effect for presentations and method demonstrations.

Download

Download the script (single-file Python tool): streamlines_tool.py

The script reads a 3D binary volume (0 = pore, 1 = solid) and a 3D Cmap (TIFF or MAT), builds a pseudo-vector field, generates streamlines inside pores, and exports an MP4 video.

How to run locally

1) Install dependencies (recommended: in a clean Python environment):

python -m pip install numpy matplotlib tifffile pillow imageio imageio-ffmpeg scikit-image scipy

2) Put your input files in the same folder as streamlines_tool.py (or edit paths in CONFIG), then run:

python streamlines_tool.py

3) Edit parameters in the CONFIG block (near the top of the script). Common settings:

  • BINARY_PATH, CMAP_PATH, OUTPUT_MP4 (input/output files)
  • VOXEL_UM (voxel size, affects aspect ratio and “boxy” appearance)
  • N_LINES, STEP, N_STEPS, MIN_POINTS (streamline density/length)
  • ENABLE_GLOW, GLOW_WINDOW, GLOW_SPEED, GLOW_SIZE (blue “comet-tail” dots)
  • DS_MESH, DS_FIELD, SMOOTH_SIGMA (speed/quality trade-offs)

Notes

  • This is a visualization demo intended for qualitative illustration (not a CFD solver).
  • The vector field is derived from the gradient of Cmap; interpretation depends on your physical model and boundary conditions.
  • If you use this demo in slides/papers, please cite your CT/transport workflow (e.g., TauFactor/Avizo/your pipeline) accordingly.
